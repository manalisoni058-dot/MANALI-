1)PURPOSE BEHIND THE RESULT : Its purpose is to generate and display the final bill of the customers fruit order ,includes the item wise cost and total amount in a clear way .
2)DATA COLLECTED FOR FRUIT ORDERING APP:
•	Fruit names (apple ,mango , banana  , etc )
•	Price of every fruit 
•	Quantity of fruit 
•	Total cost for each fruit 
•	Final total bill 

3)DATA DESCRIPTION :
•	This app uses a dictionary menu tp store fruit name as keys  and their prices as the values .
•	It also uses one more dictionary cart to store the fruits quantites selected by the user .
•	From this we can calculate :
o	Cost of each fruit = price * quantity
o	Total bill amount = sum of all items purchased by the user 
4) HOW TO USE ?
•	Run the program to see the list of fruits and prices 
•	Type name of fruit and quantity required
•	Repeat for more fruits 
•	Type done when you finish ordering .
•	The app will show the final bill and amount to be paid by the user .
